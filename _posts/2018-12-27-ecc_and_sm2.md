# 基础

椭圆曲线密码(Elliptic Curve Cryptography,ECC) 是利用椭圆曲线来实现的密码技术的统称

实际上包含三个方面

* 基于椭圆曲线的数字签名算法
* 基于椭圆曲线的密钥交换协议
* 基于椭圆曲线的公钥加密算法

椭圆曲线并不是“椭圆形”的图像。名字仅仅因为源自于求椭圆弧长的椭圆积分的反函数

一般情况下可用下列方程式来表示：
			y^2 = ax^3 + bx^2 + cx + d

例如，Fp的椭圆曲线a = 1, b = 0, 还有F2m 曲线方程为y^2 + xy = x^3 + ax^2 + b
